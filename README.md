# Atcoder_Swift
Solving Atcoder problems in Swift.  
(訳:SwiftでAtcoderの問題を解く。)
## 使用言語や技術
Swift
## ディレクトリ構造  
```
├── abc310
│   └── a
│       └── main.swift
├── abc311
│   ├── a
│   │   └── main.swift
│   └── b
│       └── main.swift
```
## 使用法
1. AtCoderで解いた問題ごとに、新しいフォルダを作成してください。
2. 問題のソースコードやメモをそのフォルダに保存します。
3. 問題に関する詳細情報をREADMEファイルに記述してください。
4. 写経した場合は、問題のタイトルの前に[!]このマークを付けて区別する。
